## 렌더 대조 중 추가로 맞춘 파리티(제거 아님, 레이아웃 정렬)
- **하단 탭바 노출 범위**: 디자인상 탭바는 **목록(`/`)·자산(`/asset`)에만** 있다. 기존 구현은 전역이라 상세·폼·백업·지출·신규 화면에도 떴다 → 목록·자산에서만 노출하도록 좁힘(`components/BottomTabBar.tsx`).
- **인증 전 화면 풀스크린**: 온보딩·잠금·로딩·에러·404는 탭바 없이 전체 높이로(디자인과 동일). `data-auth-screen`(VaultGate·not-found) + 온보딩/복구 `100dvh`.
