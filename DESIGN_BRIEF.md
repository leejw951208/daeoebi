# Secrets Manager — 화면 디자인 브리프

> 클로드 디자인 입력용 프롬프트. 인증 모델은 **passkey + WebAuthn PRF + 복구코드** 기준이다.

## 1. 제품
1인용 암호화 비밀번호 보관함. 은행·카드·증권·쇼핑 등 사이트별 비밀번호를 한곳에 암호화 보관하고, 잠깐 열람한 뒤 다시 잠근다. 공개 호스팅(Cloudflare Access가 경계 방어)이며 본인 한 명만 사용한다. 모바일 우선 PWA, 데스크탑은 가운데 정렬 phone-frame.

## 2. 사용자 · 톤
- 보안에 민감한 단일 소유자. "내 비밀이 안전하게 잠겨 있다"는 신뢰감과, 필요한 한 줄을 빠르게 꺼내 복사하고 다시 잠그는 속도가 핵심.
- 톤 — 차분하고 단단한 "금고". 화려함보다 정밀함. 과한 애니메이션 금지.

## 3. 비주얼 아이덴티티 — "Strongbox" (기존 코드와 일치 유지)
틴티드 다크 스틸 + 무광 황동(antique brass) 단일 액센트. 흔한 블루 대시보드·네온 다크와 다르게, 강철 금고 내부의 질감을 지향한다.

### 색 토큰
- Surface — `bg #12181C` · `panel #1A2228` · `plate(함몰) #0F151A`
- Border — `line #2A343C` · `strong #3A4750` · 상단 1px 금속 하이라이트 `rgba(255,255,255,.04)`
- Text — `primary #EAEFF2` · `secondary #B4C2CC` · `muted #8B9AA4`
- Accent(황동, 단일) — `#CBA45C` / hover `#E1BE76` / deep `#A8843F` / 황동 위 글자 `#14100A`
- State — danger `#F0928C` · success `#6FD2A3` · warning `#E0B452`
- 황동 틴트 — `rgba(203,164,92,.12)`, 황동 라인 `rgba(203,164,92,.32)`

### 타입 (3분할)
- 한글 산문 — Pretendard / Noto Sans KR
- 기계 라벨·숫자·eyebrow — **Space Grotesk** (대문자, letter-spacing 넓게)
- 비밀값·코드·카운트다운 — **JetBrains Mono**

### 형태 · 모션
- radius `5 / 9 / 14`, 카드는 함몰 패널 + 상단 하이라이트 엣지
- 모션은 절제. 150~250ms, ease `cubic-bezier(.2,0,0,1)`, `prefers-reduced-motion` 존중

### 시그니처 (이 페이지를 기억하게 할 단 하나)
**드러내고 스스로 사라지는 비밀 플레이트.** 민감 필드는 함몰된 모노 플레이트로, 마스킹 점 → `보기` 시 황동 모노 실제값 → `복사` 시 30초 황동 게이지가 줄며 0에서 자동 재마스킹. 자동 잠금 타이머도 같은 언어의 칩(황동→적색)으로 "노출은 시간제한"을 반복한다.

## 4. 인증 모델 — passkey + WebAuthn PRF + 복구코드
PIN·마스터 비밀번호 없음. passkey로 인증하고, WebAuthn **PRF**로 암호화 키를 도출한다(키는 서버에 저장하지 않음). 기기 분실 대비로 고엔트로피 **복구코드**를 키에 추가 래핑한다.

```
Cloudflare Access(경계)
  → passkey 인증(생체/기기) + PRF로 vault 키 도출
     → 비밀번호 목록 · 열람
        → idle N분 후 키만 메모리에서 폐기(재잠금) → passkey 재인증
```

- 최초 — passkey 등록 + 복구코드 1회 발급(반드시 저장하도록 유도)
- 재방문 — passkey 인증 한 번으로 잠금해제
- 분실 — 복구코드로 접근 복구 + 새 passkey 재등록

## 5. 그릴 화면

### 인증
1. **온보딩 / passkey 등록** — 금고 엠블럼, 제품 한 줄, "passkey 만들기" 단일 CTA. 생체/기기 인증 안내. 진행 단계 표시(등록 → 복구코드).
2. **복구코드 발급** — 고엔트로피 코드를 모노로 크게, 복사·다운로드 버튼, "안전한 곳에 보관" 경고, "저장했습니다" 확인 체크 후에야 완료.
3. **잠금해제(재방문)** — 금고 엠블럼(잠김), "passkey로 잠금해제" CTA, 생체 프롬프트 대기 상태, 실패/재시도, 대안으로 "복구코드로 접근".
4. **복구코드로 복구** — 코드 입력 필드(모노), 검증, 성공 시 새 passkey 재등록 유도.

상태 — 로딩(상태 확인 중), 에러(네트워크/인증기 미지원), rate-limit/backoff 안내.

### 보관함
5. **목록** — eyebrow `SECRETS · VAULT` + 제목, 자동 잠금 타이머 칩, 잠그기. 카테고리 필터 + 라벨 검색. 카드 스택(좌측 황동 엣지 + 카테고리 코드칩 BANK/CARD/SECURITIES/SHOPPING/OTHER). 빈 상태 = 첫 항목 추가 유도.
6. **상세** — 라벨·메타 / 민감 필드(비밀 플레이트, reveal·복사·30초 자가소멸) / 액션(수정·삭제). 카테고리별 필드 + 메모.
7. **신규 · 수정 폼** — 카테고리 선택 → 동적 필드, OTHER는 key-value(최대 10), 메모. 1차 액션 황동, 취소·삭제는 보조.
8. **카테고리 레퍼런스** — 카테고리별 필드·민감도 읽기 전용 카드.
9. **백업 · 복원** — 암호화 export 다운로드 / import(파일 + 충돌 모드 reject·skip·replace).
10. **404 / 미존재 항목** — 한국어 안내 + 복귀.

## 6. 전역 레이아웃
- 모바일 phone-frame(풀스크린), 데스크탑 가운데 768px + 상단 황동 hairline + 양옆 레터박스.
- 하단 탭바(보관함, 황동 active 인디케이터). 콘텐츠는 함몰 패널 위.

## 7. 제약 (반드시 충족)
- 접근성 — WCAG AA 대비, 키보드 포커스 가시(황동 ring), aria, 터치 타깃 44px.
- 반응형 — 모바일(≈375) → 데스크탑(≈1280) 무리 없이.
- PWA — 오프라인 셸, 다크 theme-color `#12181C`.
- 한국어 UI.
